Challenges

  1. Light Mode or Dark Mode?
    • Lots of people enjoy dark mode. Personally, I do as well, it's especially helpful during the night while my lights are dimmed. At the same time, some users do not care much for dark mode. While it's not a huge deal, I wanted to see if I can implement both color schemes without any inconveniences to the user
      • This was made possible by using 'prefers-color-scheme'
        • An issue with using this CSS media feature is the fact that it's incompatible with Internet Explorer. As a result, the default color scheme for the website is the "light" mode.
